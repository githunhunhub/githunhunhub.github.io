Markdown常用语法

发表于 2016-05-08   |   分类于 Markdown   |  

Markdown介绍

　　Markdown是一种轻量级标记语言,创始人为John Gruber。它让人以“易写易读的纯文本格式”编写文档，然后转换成有效的XHTML（或HTML）文档。它吸收了很多在电子邮件中已有的纯文本标记特性。
　　Markdown同时还是一个由Gruber编写的Prel脚本：Markdown.pl。它把用Markdown语法编写的内容转换成有效的、结构良好的HTML文档，并将左尖括号 < 和 & 替换成它们各自的字符实体引用。它可以用作单独的脚本，Blosxom和Movable Type的插件又或者BBEdit的文本过滤器。

常用语法

标题

标题分共为六级，以 # 号个数区分，所有语法后都要加空格，且用英文输入法。 

### 三级标题 
#### 四级标题 
###### 六级标题

段落和换行

首行缩进：按 `Shift` + `空格` 切换输入法为全角模式，然后按空格。
强制换行：按两下 `空格` 然后 `回车`。

强调

强调使用 * 和 _ 作为标记符号，以符号数量区分强调类型，需要前后包裹。使用如下：

_斜体_
__粗体__
___粗斜体___

区块

  • 引用区块：使用 > 标记符号，如果在列表内引用需要 Tab 缩进一次：

    > 这里是引用区块  
> This is a blockquote with two paragraphs. Lorem ipsum dolor sit amet.

  • 代码区块：多行代码使用 Tab 键缩进，如果在列表内引用，需要缩进两次：

    # -*- coding：utf-8 -*-

print('Hello, world!')

  • 行内代码区块：使用 ` ` 把代码括起来即可，如：

    `print("hello, world!")`

列表

  • 无序列表：使用 - 号或 * 号，如：

    - 无序列表1
- 无序列表2

  • 有序列表：使用 数字. 即可，如：

    1. 有序列表1
2. 有序列表2

链接

把要链接的文字用方括号 [ ] 括起来，在后边的小括号 ( ) 内插入网址，如：

[百度](http://www.baidu.com)搜索引擎。

插入图片

插入图片和插入链接类似，只需在前面多加一个感叹号，如：

![](http://www.xxx.com/ooo.jpg)
